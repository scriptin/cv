# My CV

Built with [Typst](https://typst.app/) CLI

## Building locally

- Install [Devbox](https://www.jetify.com/devbox)
- Run `devbox run watch`
- See the result in `build/`

Alternatively:

- Install Typst
- Run `typst watch --font-path src/fonts src/main.typ build/Dmitry_Shpika_CV.pdf`
- See the result in `build/`
